PHPSwitch
----------------------------------------------
PHP Version switcher for  XAMPP Windows

#### Compatibility and Requirements
* .NET Framework 4 Above
* XAMPP 5 Above
* Windows 7 Above

#### Instructions
Download PHP versions and put them in a folder following the namescheme `php_version` inside the XAMPP directory.
Examples Below:
* PHP 5.6.0 => `php_5.6.0`
* PHP 7.1.9 => `php_7.1.9`

Use the program with Administrative rights for best results
